2048 est un puzzle numérique où le joueur déplace des tuiles sur une grille de 4x4 pour les combiner.
Deux tuiles de même valeur fusionnent pour former une tuile avec leur somme.
L’objectif est d’atteindre la tuile 2048, tout en évitant que la grille ne soit complètement remplie.
Le jeu nécessite stratégie et planification pour réussir les combinaisons les plus élevées.
